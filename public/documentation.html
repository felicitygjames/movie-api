This app can be used to search for movies, directors and genres that you might be interested in. You can also save your personal information and movies you like! Enjoy!

Request Method URL Response Format
See homepage GET / Welcome to MyFlix! HTML
Get a list of all movies GET /movies None JSON
Get a specific movie title and it's information GET /movies/:Title None JSON
Get information about a specific movie's genre by it's title GET /movies/genres/:Name None JSON
Get information about a movie's director by it's title GET movies/director/:Name None JSON
Get a list of users GET /users None JSON
Add a new user POST /users User added! JSON
Lets a user update his or her information PUT /users/:Username None JSON
Lets a user add a movie to their favorites POST /users/:Username/favorites Successfully added movie to favorites! JSON
Lets a user delete a movie from their favorites DELETE /users/:Username/favorites Successfully removed movie from favorites. JSON
Shows the documentation of the app GET /documentation None JSON
