This app can be used to search for movies, directors and genres that you might be interested in. You can also save your personal information and movies you like! Enjoy!

  • /movies - returns movie list (get)
  • /genres - returns list of genres (get)
  • /directors - returns list of directors (get)
  • /users - returns list of users(get)
